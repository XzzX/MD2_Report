\chapter{Perkolation im Boolschen Modell}

Nun sollen die Perkolationseigenschaften eines porösen Mediums untersucht werden. Als Modell für das poröse Medium wird das Boolsche Modell für die Perkolation verwendet. Für die Untersuchung wird eine quadratische Simulationsbox mit der Länge $L$ verwendet. In dieser Simulationsbox werden nun Kreise mit dem Radius $R=1$ verteilt, die die Hindernisse darstellen. Eine Überlappung der Kreise ist erlaubt. Man spricht von einem perkolierenden Cluster, wenn es eine durchgängige Fläche von überlappenden Kreisen gibt, die die gegenüberliegenden Ränder verbindet. Es lässt sich zeigen, dass es ausreichend ist eine Perkolation in einer Richtung zu finden, um von allgemeiner Perkolation zu sprechen. Wäre das System unendlich vergrößert, bei gleichbleibender Dichte der Kreise, gäbe es auch in der anderen Richtung eine Perkolation.
